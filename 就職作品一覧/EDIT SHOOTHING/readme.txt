
タイトル「MapEditor」

========= オブジェクト操作 =========
オブジェクト移動->W,A,S,D
オブジェクトの高さ移動->１,２
オブジェクト変更->３,４
オブジェクト設置->Enter
オブジェクト削除->BackSpace

========= カメラ操作 =========
カメラ移動->I,J,K,L
カメラの高さ移動->U,M
注視点とカメラの距離変更->O,P
カメラの向き->十字キー
平行投影ON,OFF->９

========= セーブ＆ロード＆リセット =========
マップリセット->R
マップセーブ->F3
マップロード->F4
